== Состав компонентов

Приложение состоит из следующих компонентов:

[cols="1,1,3",options="header",width=100%]
|===
|Название|Тип|Описание

|adm-web-system-calendar-ui
|Микросервис
|Frontend-часть приложения, реализующая его веб-интерфейс.

|adm-web-system-calendar-bff
|Микросервис
|BFF-часть приложения, реализующая бизнес-логику веб-интерфейса.

|===

== Интеграция

Приложение взаимодействует со следующими внешними системами:

[cols="1,1,3",options="header",width=100%]
|===
|Название|Тип|Описание

|api-admin-storage
|Приложение
|Приложение «Карточка администратора». Передает микросервису adm-web-system-calendar-bff атрибуты администраторов.

|OPA
|Компонент с открытым исходным кодом
|Инструмент применения политик безопасности. Выполняет проверку доступа администратора к конечным точкам API микросервиса adm-web-system-calendar-bff и передает микросервису ее результаты.

|api-system-calendar
|Приложение
|Приложение «Системный календарь». Передает микросервису adm-web-system-calendar-bff данные о системных календарях и операциях над ними.

|===
